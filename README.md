## Getting Started

1. Clone the repo
   ```bash
   $ git clone https://github.com/quyencodes/nessle.git
   ```
2. Install NPM packages
   ```bash
   $ npm install
   ```
3. Run locally in development mode
   ```bash
   npm run dev
   ```
4. Project will be deployed on your browser `http://localhost:3000`
